# Ai-soc-co-pilot-project.
The AI SOC Copilot is a modular, multi-provider security tool designed to enhance Security Operations Center (SOC) efficiency by leveraging Agentic AI and Retrieval-Augmented Generation (RAG). It provides analysts with AI-driven decision support, threat intelligence mapping, and automated query suggestions.

This version is a Hardened MVP featuring mandatory API key authentication, safe vector store loading (with checksum verification), asynchronous LLM calls with timeouts, and a flexible architecture supporting Google, Groq, and Hugging Face models.
